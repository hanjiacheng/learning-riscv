## Ubuntu系统的配置

### 常用命令

```shell
sudo apt update
sudo apt upgrade
sudo apt install cmake
```

### 环境配置

```shell
export https_proxy=https://127.0.0.1:7890/
export http_proxy=http://127.0.0.1:7890/
git config --global http.proxy http://127.0.0.1:7890/
git config --global https.proxy https://127.0.0.1:7890/

sudo -E apt update
```
